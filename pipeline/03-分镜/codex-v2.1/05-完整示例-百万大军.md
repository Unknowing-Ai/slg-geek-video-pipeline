# 端到端实例：百万大军 → 完整 58s 成片

> **目的**：拿一条假设脚本，完整走完 4 步 pipeline，让你看到 codex v1.0 在实战里长什么样。
> **主题**：「为什么古代百万大军其实只有几千人？」
> **产品**：ROK 万国觉醒
> **总时长**：58s

---

## 0. 输入：v0.9.1 脚本（假设产出）

```
[Hook 段 / Curiosity / 3s]
古代史书里动不动就是百万大军，但你知道吗？真正能打的可能只有几千人。

[Core_Knowledge 段 / Intellectual_Superiority + Tension / 22s]
古代将领虚报兵力是常态，10万真兵敢号称50万。
所谓百万大军里面，有运粮民夫、辅兵、随军家属，

而一场千里远征，民夫死亡率高达60%。

[Ad_Reversal 段 / Desire_Gap / 12s]
而在万国觉醒里，你才能真正体验什么叫指挥百万雄师。
500人同屏联盟战，开局就是史诗规模。

[Selling_Point 段 / Desire_Gap + Ethnic_Superiority / 15s]

打造你的诸葛弩军，
让百万大军不再是奏报上的数字，而是你掌中真正的力量。

[CTA 段 / Desire_Gap / 6s]
现在下载万国觉醒，指挥真正属于你的百万雄师。
```

**关键标记**：
- 5 段位（无 Background，符合"少而精"原则）
- Ethnic_Superiority 只在 Selling_Point 出现 1 次（不超限）
- Tension 在 Core_Knowledge 出现（民夫死亡率 60%，弱者悲剧视角）

## 4. Step 4 输出：FFmpeg 命令序列 + 成片

按 [04-剪辑和转场.md](04-剪辑和转场.md) 的 D1-D6 模板：

### 4.1 镜头拼接

`concat_list.txt`:
```
file 'shots/01_hook_battle_far.mp4'
file 'shots/02_hook_soldier_close.mp4'
file 'shots/03_core_official_writing.mp4'
... (25 个)
```

```bash
ffmpeg -f concat -safe 0 -i concat_list.txt -c copy step1_concat.mp4
```

### 4.2 Ad_Reversal 段位转场（插入黑屏 0.3s 在镜头 12 后）

```bash
# 生成 0.3s 黑屏
ffmpeg -f lavfi -i color=black:size=1080x1920:duration=0.3 \
       -f lavfi -i anullsrc=channel_layout=stereo:sample_rate=44100 -t 0.3 \
       -c:v libx264 -pix_fmt yuv420p -c:a aac black_03s.mp4

# transition_list.txt 改为：
# file 'shots/12_core_warfare_slow.mp4'
# file 'black_03s.mp4'                ← 新插入
# file 'shots/14_ad_logo.mp4'

ffmpeg -f concat -safe 0 -i transition_list.txt -c copy step2_transition.mp4
```

### 4.3 字幕烧录（ASS 文件）

```bash
ffmpeg -i step2_transition.mp4 -vf "ass=subtitle.ass" -c:a copy step3_subtitle.mp4
```

### 4.4 SFX 合成

```bash
ffmpeg -i sfx/boom.wav -i sfx/ding.wav -i sfx/whoosh.wav -i sfx/glitch.wav -i sfx/cash.wav \
       -filter_complex "[0:a]adelay=2500|2500[boom1]; \
                        [1:a]adelay=15500|15500[ding1]; \
                        [1:a]adelay=17500|17500[ding2]; \
                        [3:a]adelay=25300|25300[glitch1]; \
                        [0:a]adelay=25300|25300[boom2]; \
                        [2:a]adelay=25300|25300[whoosh1]; \
                        ... [所有 SFX adelay]; \
                        [boom1][ding1][ding2][glitch1][boom2][whoosh1]...amix=inputs=15:duration=longest[sfx_combined]" \
       sfx_track.wav
```

### 4.5 三轨混音

```bash
ffmpeg -i step3_subtitle.mp4 -i voiceover.wav -i bgm_history_epic.mp3 -i sfx_track.wav \
       -filter_complex "[1:a]volume=1.0[voice]; \
                        [2:a]volume=0.15[bgm]; \
                        [3:a]volume=0.6[sfx]; \
                        [voice][bgm][sfx]amix=inputs=3:duration=longest[aout]" \
       -map 0:v -map "[aout]" -c:v copy -c:a aac step4_mixed.mp4
```

### 4.6 整体调色（按段位渐变）

```bash
# Hook 段 0-3s 偏暗冷
# Core_K 段 3-25s 中性偏暗
# Ad_R 段 25.3-37s 突亮高饱和
# Selling 段 37-52s 高饱和暖色
# CTA 段 52-58s 暖收束

ffmpeg -i step4_mixed.mp4 \
       -vf "split=5[v1][v2][v3][v4][v5]; \
            [v1]trim=0:3,eq=saturation=0.9:contrast=1.05[hook]; \
            [v2]trim=3:25,eq=saturation=0.95[core]; \
            [v3]trim=25.3:37,eq=saturation=1.2:contrast=1.15[adr]; \
            [v4]trim=37:52,eq=saturation=1.15:contrast=1.1[sell]; \
            [v5]trim=52:58,eq=saturation=1.1:contrast=1.2[cta]; \
            [hook][core][adr][sell][cta]concat=n=5" \
       -c:a copy step5_graded.mp4
```

### 4.7 最终输出（投放格式）

```bash
ffmpeg -i step5_graded.mp4 -c:v libx264 -preset slow -crf 23 \
       -c:a aac -b:a 192k -movflags +faststart \
       output_百万大军_v1.mp4
```
