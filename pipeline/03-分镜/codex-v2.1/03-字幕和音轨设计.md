# Step 3: 口播 → 字幕 + 音轨

> **输入**：脚本的口播文案 + 段位类型 + 分镜表
> **输出**：字幕样式稿（位置/颜色/动画）+ BGM 选型 + SFX 卡点表（精确到毫秒）
> **谁做**：剪辑师 / 自动工具（FFmpeg + ASS 字幕脚本）
> **关键约束**：字幕颜色三色体系 / Ad_Reversal 必 BGM 突变 / SFX 毫秒级卡点

---

## A. 字幕设计

### A1. 三色语法（SUB-01）

| 颜色 | hex | 用于段位 | 含义 |
|---|---|---|---|
| **白色** | #FFFFFF | Hook + Core_Knowledge | 信息色 / 知识色（中性） |
| **金色** | #FFD700 | Ad_Reversal + Selling_Point + CTA | 价值色 / 行动色（商业关键） |
| **红色** | #FF0000 | Hook 关键词点缀 | 警示色 / 冲击色（强反差） |

**红线**：
- 单视频颜色种类上限 3 种（白/金/红），禁用其他装饰色
- 同一段位内禁混 3 种以上颜色

### A2. 位置坐标系

视频画幅 9:16（竖屏 1080×1920），位置用 x_percent / y_percent 表示（0=左上，100=右下）：

| 字幕类型 | 推荐位置 | 用途 |
|---|---|---|
| 顶部大字标题 | x=50, y=15 | Hook 段疑问句 / Selling_Point 卖点总结 |
| 屏幕中央强调字 | x=50, y=50 | Ad_Reversal 反转词 / CTA 行动指令 |
| 底部同步字幕 | x=50, y=85 | 跟随口播的逐字字幕（最常用） |
| 左下角标签 | x=15, y=90 | 时间戳 / 注释（极少用） |

**安全区**：避开顶部 220px UI 区 + 底部 400px 操作区（按钮区）

### A3. 字号与字体

| 段位 | 字号（相对画面高度）| 字体 |
|---|---|---|
| Hook 顶部大字 | 8-10%（约 150-200px） | 思源黑体 Bold |
| Core_Knowledge 同步字幕 | 4-5%（约 80-100px） | 思源黑体 Medium |
| Ad_Reversal 中央强调 | 8-12%（约 150-230px） | 思源黑体 Bold |
| Selling_Point 顶部 | 6-7%（约 110-140px） | 思源黑体 Bold |
| CTA 中央 | 7-9%（约 140-170px） | 思源黑体 Bold |

### A4. 动画类型

| 动画 | 用途 | 触发场景 |
|---|---|---|
| **Pop_Up**（弹出）| 关键数字 / 反转词 / 产品名 | 字幕首次出现时 |
| **Shake**（抖动）| 强冲击词 / 警示词 | Hook 红色高亮字 |
| **None**（静态）| 普通同步字幕 | Core_Knowledge 底部字幕 |
| **Fade_In**（渐入）| 段位首字幕 | 段位切换时 |

### A5. 字幕节奏

- **单屏字数上限**：12-15 字（超过会让人来不及读）
- **字幕停留时长**：与口播严格同步，每个字幕 ≥ 1.5s（短于 1.5s 看不清）
- **关键词加粗 / 变色 / 放大**：每段不超过 2 个，过多失去重点

## B. BGM 设计

### B1. 全片 BGM 风格（看主题决定）

| 题材 | BGM 风格 | 参考曲库 |
|---|---|---|
| 历史军事（古代战争）| 战鼓 + 史诗交响 | Epidemic Sound 史诗类 |
| 现代军事（武器装备）| 紧张电子 + 金属打击 | Artlist 军事类 |
| 动物 / 生态 | 悬疑铺垫 + 自然环境音 | Audio Network 自然类 |
| 文化 / 文明 | 古风民乐 + 西方交响混合 | Musicbed 文化类 |
| 奇幻 / RPG | 史诗交响 + 合成器 | Epidemic Sound 奇幻类 |

### B2. 段位 BGM 控制（BGM-01/02/03）

| 段位切换 | BGM 操作 | 突变率（数据） |
|---|---|---|
| 开场（0s）| 从静音渐入战鼓 / 悬疑音 | — |
| Hook → Background/Core_K | 平稳过渡 | 22% |
| Core_K → Ad_Reversal | **必突变：戛然而止 + 黑屏 0.3s + 史诗起势** | **50%+** |
| Background → Ad_Reversal | **必突变：同上** | **61.5%** |
| Ad_Reversal → Selling_Point | 延续史诗音乐，节奏加快 | 16% |
| **Selling_Point → CTA** | **不突变（保持高潮延续，禁止断裂）** | **15.3%** |
| 结尾（最后 1s）| 推向最高潮并定格 | — |

### B3. 音量曲线（口播 vs BGM 平衡）

| 段位 | 口播 dB | BGM dB | 说明 |
|---|---|---|---|
| 全程 | -6 dB | -18 dB | 口播段 BGM 降 12dB |
| 无口播段（Hook 前 0.5s / 转场）| — | -10 dB | BGM 升起来填空 |
| 关键反转（Ad_Reversal 起点）| — | -3 dB | BGM 短暂拉高制造冲击 |

## C. SFX 音效卡点

音效是极客号视频的"听觉灵魂"。SFX 必须精确到毫秒，与画面/字幕同步。

### C1. SFX 类型库

| 类型 | 触发场景 | 时长 | 典型音色 |
|---|---|---|---|
| **Heavy_Boom** | 冲击词 / 关键数据 / LOGO 出现 | 0.3-0.5s | 重低音爆炸 / 鼓击 |
| **Ding** | 数字 / 名将出现 / 列表项 | 0.2-0.3s | 清脆金属敲击 |
| **Whoosh** | 转场 / 物体快速移动 | 0.5-1s | 风掠过 |
| **Glitch** | 段位切换（特别 Ad_Reversal）| 0.3-0.5s | 故障扭曲音 |
| **Cash_Register** | 卖点 / 数字飙升 | 0.5s | 收银机声 |
| **Static**（静音）| 关键反转前 0.5-1s | — | 制造听觉真空 |

### C2. SFX 卡点表模板

每条视频做一份 SFX 卡点表：

| 时间 (ms) | 镜头# | 关联字幕/词 | 音效类型 | 时长 | 备注 |
|---|---|---|---|---|---|
| 2500 | 1 | "百万" Pop_Up | Heavy_Boom | 0.4s | Hook 关键词冲击 |
| 18000 | 7 | "60%" 数字出现 | Ding | 0.3s | 数据可视化 |
| 25000 | 11 | 黑屏切换瞬间 | Static | 0.3s | 静音真空（Ad_R 前） |
| 25300 | 12 | LOGO Pop_Up | Heavy_Boom + Whoosh | 0.5s | 段位切换冲击 |
| 30000 | 14 | "百万雄师" 字幕 | Ding × 3 | 0.6s | 卖点列表 |
| 55000 | 24 | CTA "现在下载" | Cash_Register + Heavy_Boom | 0.5s | 收尾强调 |

### C3. SFX 卡点黄金规则

1. **字幕 Pop_Up 必带 SFX**：每个 Pop_Up 字幕必须配 Ding 或 Heavy_Boom
2. **段位切换必带 SFX**：Ad_Reversal 起点必须有 Glitch / Whoosh / Heavy_Boom 组合
3. **数字出现必带 Ding**：所有数据可视化的关键数字都要配 Ding
4. **反转前留静音**：关键反转前 0.3-0.5s 全静音（包括 BGM），制造期待感
5. **CTA 段密集 SFX**：CTA 最后 3s 内至少 2 个 SFX（Heavy_Boom + Cash_Register）

## D. ASS 字幕脚本模板

ASS（Advanced SubStation Alpha）是支持位置/颜色/动画的字幕格式，FFmpeg 可直接烧录。

```ass
[Script Info]
ScriptType: v4.00+
PlayResX: 1080
PlayResY: 1920

[V4+ Styles]
Format: Name, Fontname, Fontsize, PrimaryColour, OutlineColour, Bold, Alignment, MarginL, MarginR, MarginV
Style: HookTitle,思源黑体 Bold,180,&H00FFFFFF,&H00000000,1,8,0,0,288
Style: HookHighlight,思源黑体 Bold,180,&H000000FF,&H00000000,1,8,0,0,288
Style: SyncBottom,思源黑体 Medium,90,&H00FFFFFF,&H00000000,0,2,0,0,1632
Style: AdReversal,思源黑体 Bold,200,&H0000D7FF,&H00000000,1,5,0,0,0
Style: CTABottom,思源黑体 Bold,160,&H0000D7FF,&H00000000,1,5,0,0,0

[Events]
Format: Layer, Start, End, Style, Name, MarginL, MarginR, MarginV, Effect, Text
; Hook 段：白底标题 + 红色"百万"高亮
Dialogue: 0,0:00:00.00,0:00:03.00,HookTitle,,0,0,0,,{\an8}{\fad(300,0)}百万大军，其实只有几千？
Dialogue: 1,0:00:00.30,0:00:03.00,HookHighlight,,0,0,0,,{\an8}{\fad(300,0)}{\t(\fscx120\fscy120)}百万

; Core_Knowledge 段：底部同步字幕
Dialogue: 0,0:00:03.00,0:00:08.00,SyncBottom,,0,0,0,,古代将领虚报兵力是常态
Dialogue: 0,0:00:08.00,0:00:14.00,SyncBottom,,0,0,0,,所谓百万大军里，运粮民夫占大半
```

**关键 ASS 标签**：
- `{\an8}` = 顶部居中 / `{\an5}` = 屏幕中央 / `{\an2}` = 底部居中
- `{\fad(300,0)}` = 渐入 300ms
- `{\t(\fscx120\fscy120)}` = 字号 1.2 倍放大动画（Pop_Up 效果）
- `{\t(\frx10)}` = 抖动（旋转 10 度）→ Shake 效果
