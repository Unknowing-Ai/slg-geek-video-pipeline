## B · 5 类 Hook 类型库

### B-1 揭秘式（"为什么…"类）

**机制**：抛一个反常识的"为什么"问题，让用户停下来想"对啊，为什么呢"。

**句式模板**：
- "为什么 [常见现象]？因为 [反常识答案]"
- "你以为 [常见认知]，其实 [真相]"
- "[某事物] 到底有多 [极端形容]？"

**正反对照**（关键，别学错）：

| ✗ 平庸 Hook | ✓ 极客式 Hook | 区别 |
|---|---|---|
| "以步制骑" | "为什么只有中国人才能以步制骑" | 前者是裸知识点，后者带"独有 / 我们 vs 他们"情绪框架 |
| "驱逐舰很猛" | "为什么说驱逐舰是潜艇的克星" | 前者是事实陈述，后者是反差判断 + 留悬念 |
| "古代打仗很难" | "为什么古代士兵明明每天都要吃饭、一打仗就缺粮" | 前者抽象空话，后者具体悖论 |

**铁律**：揭秘式 Hook **必须自带情绪框架的种子**（民族优越感 / 智识优越感 / 欲望落差），不能是中性知识点。

### B-2 反转式（"你以为…其实"类）

**机制**：先肯定一个常见认知，再反转打破。

**句式模板**：
- "你以为这是 X，其实是 Y"
- "有人问 [问题]…仔细看 [证据] 就明白"（先假装否定后反转肯定）
- "看似 [印象]，实则 [真相]"

**样本**：WGAME_11 轰炸机瞄准器 / Samo_21 兽族空军 / Samo_23 驯化老虎

### B-3 极端式（"最 / 史上 / 竟然"类）

**机制**：用一个极端对比/数字制造视觉震撼。

**句式模板**：
- "[事物] 出现的时候，[反差锚点] 还没 [常见状态]"（例："它问世的时候，我们村甚至还没有通电"）
- "仅 [数量] [事物]，就足以 [毁灭性结果]"
- "[事物] 服役 [极长时间]，至今难以被超越"

**样本**：WGAME_13 阿帕奇（"我们村还没通电"是教科书级极端式 Hook）

### B-4 数字式（"X 倍 / X 万 / X 美元"类）

**机制**：用具体数字让抽象的猛/贵/强变得可感知。

**句式模板**：
- "[功能] 提升 X%，[效果]"
- "造价高达 X 美元 / 射速 X 发每分钟 / 杀敌 X 万"

**注意**：在极客号样本里，数字式很少做主 Hook，更多是嵌入在 B-1/B-2 的展开段里强化震撼。

### B-5 类比式（古今对照 / 跨域对比）

**机制**：把古代/异域/动物的事和现代/中国/人类做对比，制造熟悉感+距离感的双重作用。

**句式模板**：
- "[古代场景]，[现代视角] 直到 X 年才学会"
- "[他们] 把 [东方/我们] 的东西称为 [洋词]"（例："欧洲人称蒙古铁骑为'上帝之鞭'"）
- "如果 [古代人] 在一个时代会怎样"

**样本**：ROK_05 东西方骑兵对比 / ROK_02 世界强者齐聚（古今穿越脑洞）

### B-Hook 类型分布（n=75）

| Hook 类型 | 头部 | 中间 | 尾部 | 总占比 |
|---|---|---|---|---|
| 悬念问题（B-1+B-2） | 83% | 71% | 42% | 67% |
| 对比反差（B-5） | 11% | 7% | 26% | 13% |
| 数据/视觉轰炸（B-3+B-4） | 5% | 5% | 10% | 7% |
| 异常视觉 | 0% | 10% | 15% | 9% |

**结论**：B-1 揭秘式是该号 Hook 主形态（占 2/3），B-2 反转和 B-5 类比是次主形态，B-3/B-4 是辅助技法（嵌入展开段）。

## C · 7 段位结构定义

| 段位 | 时间占比 | 时长（50s 视频） | 口播字数 | 画面 | 核心任务 |
|---|---|---|---|---|---|
| 1 Hook | 8% | 0-4s | 20-35 | 真世界 75% | 抛悬念/反差，建立情绪种子 |
| 2 铺垫A | 14% | 4-14s | 30-55 | 真世界 75% | 用 1 个证据/数据展开 Hook，深化知识感 |
| 3 铺垫B | 14% | 14-26s | 35-60 | 真世界为主 | 第 2 个证据 + 情感峰值（卡在最爽处） |
| 4 桥梁 | 14% | 26-33s | 35-60 | 首次切游戏 | 一句话从真世界跳到游戏里（用"然而 / 而今 / 但是在 X 里"） |
| 5 玩法A | 14% | 33-41s | 35-55 | 游戏 100% | 因果回答 Hook：在游戏里如何解决/重现/碾压 |
| 6 玩法B | 13% | 41-47s | 30-50 | 游戏 89% | 在玩法A 上做**维度跳跃**（不是平行展示第二卖点） |
| 7 CTA | 7% | 47-50s | 15-25 | 游戏+Logo | 召唤行动，含产品名 |

**字数容忍度**：±10% 浮动可接受，不要硬卡死。

**节奏要点**：
- 0-4s 必须在 75% 画面内完成情绪种子（图像震撼 + 第一句口播抛出悬念词）
- 25-30s 是桥梁转折点，BGM 在此处必须有变化
- 40-47s 是玩法 B 的维度跳跃区，必须出现"不止…还有 / 更绝的是 / 真正的 / 一体化"等扩展词

## D · 9 类词库（极客号语料）

```

```

```
殊不知 / 别说 / 抛开…不说 / 不仅…还
```

### D-3 桥梁过渡句


## G · 情绪燃料三选一定义

每条脚本生成时**必须显式选择 1 个主燃料**（也可叠加，但有主有次）。情绪燃料是 Hook 设计的**起点**，不是事后加层。

### G-1 民族优越感

**核心叙事**："我们 vs 他们" / 中国 vs 西方 / 东方智慧 vs 蛮力。

**Hook 触发词**：
```
为什么只有 [中国/我们] 才能…
[他们] 直到 X 年才学会，我们 [古代时间] 就在用
[中国名将] 让 [外国对手] 大为震惊
古代中国 vs 西方军事差距
```

**注意**：不是单纯讲中国厉害，必须有"对照系"——和谁比、比的什么、谁赢了。

### G-2 智识优越感

**核心叙事**：你不知道的我知道 / 看似普通其实有玄机。

**Hook 触发词**：
```
你以为这是 X，其实是 Y
为什么 [反常识现象]
[军事/科学概念] 的真相是…
真正懂行的人都知道…
你见过 [极端反差现象] 吗
```

### G-3 欲望落差

**核心叙事**：现实做不到 → 游戏里能做到 / 现实有限制 → 游戏里无限制。

**Hook 触发词**：
```
为什么人类没有 [驯化/拥有/做到] X
现实中不可能的 X，在 [游戏] 里…
[古代/动物/异世界] 的 X，你也能 [操控/驾驭/重现]
有人问为什么 [反常现象]…仔细看就明白
```

## H · 因果路径模板

**定义**：从 Hook 的现实痛点 → 卖点机制 → 玩家可感知结果，必须是因果链条，不是话题贴边。

### H-1 模板（必须严格对应 Hook 痛点）

```
[Hook 现实痛点：例 古代缺粮]
↓
[卖点机制：例 中式文明 5% 建造加速 + 高筑墙广积粮天赋]
↓
[玩家可感知结果：例 别人慢慢发育时你已经百万诸葛弩兵临城下]
```

### H-2 真伪检验（自查）

**伪因果路径**（必须 FAIL）：

| Hook 痛点 | 玩法卖点 | 问题 |
|---|---|---|
| 古代缺粮 | "你能操控火炮单位" | 风马牛不相及 |
| 古代缺粮 | "资源补给系统让你不缺粮" | 该游戏没有这个系统，AI 编出来的 |
| 古代缺粮 | "丰富的兵种" | 抽象空话，没有解决"缺粮"这个具体痛点 |

**真因果路径**（PASS）：

| Hook 痛点 | 玩法卖点 | 玩家感受 |
|---|---|---|
| 古代缺粮 | 中式文明 5% 建造加速 + 诸葛弩快速解锁 | "我比别人快、所以我不会被缺粮拖垮" |
| 驯化不了老虎 | 40 人巨兽团本团队压制烈焰巨龙 | "现实不行的，我在游戏里能做到" |
| 驱逐舰克潜艇 | 200+ 现代军武 + 海陆空协同 | "现实里的硬核，在游戏里能亲自指挥" |

### H-3 换名检验（最重要）

把脚本里产品名替换成竞品名（例万国觉醒 → 列王的纷争），如果**整段读下来仍然顺畅**，说明卖点没绑定该产品的独特机制，必须 FAIL 重写。

例：
- ✗ "在 X 里你能驾驭巨兽" → 换成任何竞品也成立 → FAIL
- ✓ "在万龙觉醒的 40 人团本里，4 个职业分工击败烈焰巨龙" → 这是该产品独有的具体机制 → PASS
