# 04 · 校准层（Few-shot Examples）

**作用**：让模型从真实极客号头部样本里学到具体的文字质感、节奏感、情绪燃料嵌入手法。

**结构**：Tier 1 三条核心样本（按段位逐句标注）+ Tier 2 三条扩展样本（关键拆解）。共 6 条覆盖三个产品 + 三种情绪燃料 + 5 类 Hook 主类型。

**使用方式**：模型在 04 节读完后，应当能在脑子里建立"原来该号是这样写'刘彻看着对面的罗马军团仅仅大手一挥'这种句子的"的感觉。这种感觉无法通过 do/don't 列表传递，只能通过样本传递。

**最关键警告**：04 是用来让你**学风格质感**，不是用来让你**逐字抄 Hook**。每条 Tier 1 样本末尾有一个"防过拟合提醒"块，明确告诉你这条样本里哪些可以抽象复用 / 哪些是该样本独有不可照搬。生成新脚本时，与样本 Hook 字面重合度必须 ≤ 40%（见 03-style-guide Step 5 + 自审表 8）。

### 样本 1 · ROK_头_01 为何古代一打仗就缺粮

**元信息**

| 项 | 值 |
|---|---|
| 产品 | 万国觉醒 |
| 时长 | 50.0 秒 |
| CTR(7日) | 0.50% |
| 消耗 | ¥965,737.66 |
| 主情绪燃料 | 民族优越感（中式文明战争智慧） |
| Hook 类型 | B-1 揭秘式（反常识悖论） |
| 具体层主线词 | 古代缺粮 / 兵马未动粮草先行 |
| 抽象层主线词 | 中国人的战争后勤智慧 |

**逐段标注**

#### 段 1 · Hook（0-4.5s）

> **口播**："古代士兵明明每天都要吃饭，为何一打仗就缺粮？因为粮草运输难度太大了。"
>
> **字幕**：古代士兵明明每天都要吃饭为何一打仗就缺粮 / 因为粮草运输难度太大了

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 钩子词"为何"打头 / ✅ 反常识悖论结构（每天吃饭 vs 一打仗就缺粮）/ ✅ 不到 35 字 / ✅ 真世界画面 |
| 词库标签 | D-1 钩子词：为何 / D-9 视觉：影视混剪、士兵方阵 |
| 燃料嵌入 | 智识优越感种子（你没想过的问题，但答案有道理） |
| 风格质感学习点 | "明明…为何…"是该号最经典的悖论 Hook 结构。Hook 不仅抛问题，还**给了一个简短答案**（"运输难度太大"），但留下"具体多难"的悬念给铺垫段展开 |

#### 段 2 · 铺垫A（4.5-12s）

> **口播**："古代运粮需要大量的人马和时间，途中就会消耗掉大半粮食。更别说遇到敌人偷袭等突发事件，真正送达前线的十不存一。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 给出第一证据（路上消耗）/ ✅ 数字感（"大半"/"十不存一"）/ ✅ 真世界画面（运粮独轮车、夜间运粮）/ ✅ ~50 字 |
| 词库标签 | D-2 铺垫连接词：更别说 / D-8 数字句式：十不存一 |
| 燃料嵌入 | 智识感（用户在了解"运粮"这个事的真实复杂度） |
| 风格质感学习点 | "十不存一"是有杀伤力的成语，比"大部分都浪费了"强 10 倍。该号铺垫段擅长用**短而狠的成语**强化数据 |

#### 段 3 · 铺垫B（12-26.5s）

> **口播**："其次打仗时士兵体力消耗巨大，吃的自然也比平时要多。战马更可怕，一匹马吃的比三个兵还多。再加上打仗时男人都上了战场，劳动力减少，粮食产量自然大大降低。所谓兵马未动，粮草先行。粮草就是军队的命脉，没粮再猛的兵也得崩。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 给出第二、第三证据（士兵食量 / 战马食量 / 男人上战场）/ ✅ 情绪推到峰值（"粮草是命脉，没粮再猛的兵也得崩"）/ ✅ 长段（90+ 字，但分多个短句）/ ✅ 真世界画面+情绪高潮（士兵推翻瓦罐） |
| 词库标签 | D-2 连接词：其次/再加上 / D-8 数字句式："一匹马吃的比三个兵还多" |
| 燃料嵌入 | 智识优越感（多层因果链让用户感到"原来这么复杂"）+ 民族文化感（"兵马未动粮草先行"古训） |
| 风格质感学习点 | "一匹马吃的比三个兵还多"是教科书级**具象数字**——不说"消耗大"，说"3 倍"。该号铺垫段必有 1 个让用户能记住的具体对比 |

#### 段 4 · 桥梁（26.5-35s）

> **口播**："然而在万国觉醒中，你完全不用担心这个问题。只要开局选择中式文明，利用百分之五的建造加速天赋，让你快速成建发展科技，高筑墙广积粮。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 标准桥梁句"然而在 [产品] 中" / ✅ 一句话切游戏画面 / ✅ BGM 转折（沉重 → 激昂） |
| 词库标签 | D-3 桥梁过渡句：然而在 X 中 / D-4 玩法引入：让你 |
| 因果路径 | 现实痛点（缺粮）→ 卖点机制（中式文明 5% 建造加速 + 高筑墙广积粮天赋）→ 玩家结果（你不用担心这问题） |
| 风格质感学习点 | "高筑墙广积粮"是**化用古代术语进游戏卖点**的精妙写法——既保民族文化感，又是具体游戏机制。该号桥梁最擅长这种"文言金句 + 游戏机制"组合 |

#### 段 5 · 玩法A（35-40.5s）

> **口播**："早早解锁远程利器诸葛弩。别人还在慢慢发育，你的百万诸葛弩军早已兵临城下。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 1 个聚焦卖点（诸葛弩）/ ✅ 因果回答 Hook（不缺粮 → 快速发展 → 提前解锁诸葛弩）/ ✅ 玩家可感知动作（兵临城下） |
| 词库标签 | D-4 玩法引入：早早解锁 / 你的 |
| 维度对比锚点 | "别人还在慢慢发育，你..."（建立比较） |
| 风格质感学习点 | "百万诸葛弩军"——具象数字 + 兵种名 + 动作（兵临城下）三件套。比"军力强大"具体 100 倍 |

#### 段 6 · 玩法B（40.5-47s）

> **口播**："何况还有始皇嬴政、杀神白起、冠军侯霍去病等众多强力统帅，助你横扫战场，一统万国更是轻轻松松。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 维度跳跃（玩法A 单兵种 → 玩法B 名将群+终极目标）/ ✅ 跳跃方向：个体能力 → 体系感（多名将组成的中式文明统帅体系） |
| 词库标签 | D-2：何况还有 / D-5 升级词：横扫、一统 |
| 燃料嵌入峰值 | 民族优越感峰值（始皇/杀神/冠军侯三个民族 icon 排比） |
| 风格质感学习点 | "始皇嬴政、杀神白起、冠军侯霍去病"——名将的"专属修饰词"是民族叙事关键。不说"嬴政白起霍去病"，说"始皇嬴政、杀神白起、冠军侯霍去病"。每个名将带一个民族符号 |

#### 段 7 · CTA（47-50s）

> **口播**："快来万国觉醒，驰骋沙场，征服万国吧。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 含产品名 / ✅ 动词（快来/驰骋/征服）/ ✅ 13 字（≤25）/ ✅ 不含空话 |
| 词库标签 | D-6 CTA 动词：快来 / D-7 CTA 模板：快来 [产品名] [核心体验] |
| 风格质感学习点 | CTA 不堆砌——3 个动词足够。"驰骋沙场，征服万国"用对仗式短句，比"开始你的战斗之旅"有力 |

**整篇关键学习点**

1. Hook 用"明明…为何…"的悖论结构是该号最经典套路之一
2. 铺垫段必有 1 个具体数字对比（一匹马吃 3 个兵的粮）
3. 桥梁段用"然而在 X 中"一秒切游戏，配合 BGM 转折
4. 化用古代术语进游戏卖点（"高筑墙广积粮"= 中式 5% 建造）
5. 名将必带专属修饰词（始皇/杀神/冠军侯）强化民族 icon

**⚠️ 防过拟合提醒（这条样本你应该学什么 / 不应该照搬什么）**

| 可以抽象复用（学风格） | 绝对不可以照搬（这是该样本独有） |
|---|---|
| 句式骨架：钩子词"为何"+ 悖论结构（A 现象 vs 反 A 结果） | Hook 原句"古代士兵明明每天都要吃饭，为何一打仗就缺粮"——直接复用 = FAIL |
| 桥梁公式："然而在 [产品] 中" | "高筑墙广积粮"作为卖点包装手法可学（化用古训词），但这个具体词条不要重复使用 |
| 数字对比手法（一匹马 = 3 个兵的粮） | 具体数字"5% 建造"是万国卖点；其他产品不要张冠李戴 |
| 名将+修饰词组合（XX+icon） | "始皇嬴政、杀神白起、冠军侯霍去病"原句不要照搬 |

**生成新 Hook 时**：换"为何 X 明明 Y 却 Z"的句型框架，**主语 / 现象 / 反差点全部换成新内容**。例：
- ✅ "为何古代行军那么慢，独独中国人能日行八百"
- ✅ "为何长城修了两千年，秦汉一砖只墙就能扛住外族"
- ✗ "古代将军明明每天都在训练，为何一打仗就溃败"（句式抄太死）

### 样本 4 · WGAME_头_13 阿帕奇有多猛

**元信息**

| 项 | 值 |
|---|---|
| 产品 | 战火勋章 |
| 时长 | 55.0 秒 |
| 消耗 | ¥253,902.12 |
| 主情绪燃料 | 智识优越感 + 极端值震撼 |
| Hook 类型 | B-3 极端式 + B-4 数字式 |
| 具体层主线词 | 阿帕奇直升机 |
| 抽象层主线词 | 跨越时代的绝对压制力 |

**逐段标注**

#### 段 1 · Hook（0-4.5s）

> **口播**："它问世的时候，我们村甚至还没有通电。阿帕奇武装直升机到底有多猛？当它在 1986 年首次呼啸升空时，"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 极端式 Hook 教科书 / ✅ "它问世的时候，我们村甚至还没有通电"是**反差锚点**——把抽象的"先进"用具体的"我们村没通电"表达 / ✅ 真世界画面（阿帕奇飞行录像） |
| 词库标签 | D-1：到底有多 / D-8 数字句式：1986 年 |
| 燃料嵌入 | 智识感峰值（你不知道阿帕奇有多老 + 中国发展史的代差对比） |
| 风格质感学习点 | **该号极端式 Hook 的最高模板**：找一个"低维度反差锚点"（我们村没通电）来衬托"高维度事物"（阿帕奇）。这种反差比直说"很先进"震撼 1000 倍 |

#### 段 2 · 铺垫A（4.5-12.5s）

> **口播**："对于全球众多军队而言，它就像是来自未来的武器。服役四十载，至今难以被超越。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 给出第一证据（全球军队认知 + 40 年服役期）/ ✅ "来自未来的武器"是金句 |
| 词库标签 | D-8：服役四十载 |
| 燃料嵌入 | 智识感持续 |
| 风格质感学习点 | "来自未来的武器" = 把武器拟人化为"穿越者"。比"先进性"强很多 |

#### 段 3 · 铺垫B（12.5-27s）

> **口播**："一门射速每分钟 625 发的 30 毫米链式机炮，16 枚地狱火反坦克导弹，76 枚 70 毫米火箭弹，搭配长弓毫米波雷达，以及看哪打哪的独门绝技。毫不夸张的说，仅出动一架阿帕奇，就足以摧毁一个坦克加强连。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ **数据轰炸教科书**：4 个具体武器配置 + 1 个夸张结论 / ✅ 情绪峰值（"摧毁一个坦克加强连"）/ ✅ 真世界画面（武器特写、装填、爆炸） |
| 词库标签 | D-8 数字句式：射速每分钟 X 发 / X 枚 Y 导弹 / "仅 X 就足以 Y" |
| 燃料嵌入 | 智识感峰值（具体到每分钟射速、导弹数量、弹药口径——硬核到极致） |
| 风格质感学习点 | "仅出动一架阿帕奇，就足以摧毁一个坦克加强连"——具体小数字 vs 具体大数字（1 vs 一个加强连=约 17 辆坦克）的对比是数字式 Hook 必备 |

#### 段 4 · 桥梁（27-31.7s）

> **口播**："而今在战火勋章中，这款传奇武直，也被一比一真实还原在你的掌中。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 标准桥梁句"而今在 [产品] 中" / ✅ "一比一真实还原"+"在你的掌中"是"现实硬核 → 你能操控"的丝滑过渡 |
| 词库标签 | D-3：而今在 X 中 |
| 因果路径 | Hook 痛点（阿帕奇这么猛你只能远观）→ 卖点机制（游戏 1:1 还原）→ 玩家结果（你能亲自操控） |
| 风格质感学习点 | "在你的掌中"——把抽象的"游戏体验"具象为"掌中"，比"游戏内还原"更有触感 |

#### 段 5 · 玩法A（31.7-44s）

> **口播**："从它猎杀级的机动性，到凶悍的火力配置，从独特的生存设计，到人机合一的战术逻辑，都将在你指尖完美重现。你可以亲自指挥阿帕奇直升机编队，体验这份跨越时代的绝对压制力。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 因果回答（前面讲阿帕奇 4 项硬核参数，这里讲游戏里都能玩到）/ ✅ 卖点真实（编队指挥）/ ✅ 主线词回响（"跨越时代的绝对压制力"= 抽象层主线词） |
| 词库标签 | D-4：你可以亲自指挥 |
| 风格质感学习点 | "从…到…，从…到…，都将…"——平行排比把铺垫段的硬核要点逐一对应到游戏里。**铺垫和玩法 A 必须有 1:1 的"问答对应感"** |

#### 段 6 · 玩法B（44-49.3s）

> **口播**："你甚至可以换装 127 毫米火箭巢，涡轴发动机等配件，打造属于你的最强武直。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 维度跳跃（玩法A 操控编队 → 玩法B 改装配件，从"指挥"到"个性化定制"）/ ✅ 跳跃方向：个体能力 → 体系感（武器配置系统） |
| 词库标签 | D-4：你甚至可以 / D-8 数字：127 毫米 |
| 风格质感学习点 | "你甚至可以"——"甚至"是升级感的标志词。让用户感到"还有这种操作？" |

#### 段 7 · CTA（49.3-55s）

> **口播**："让敌人感受被低空死神支配的恐惧。快来战火勋章打一场爽快的现代战争。"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 含产品名 / ✅ 动词（快来、打）/ ✅ "低空死神"是燃料尾声 / ✅ 25 字内 |
| 词库标签 | D-6：快来 |
| 风格质感学习点 | "低空死神支配的恐惧"——CTA 前加一句"杀伤力收尾"句，让燃料情绪不掉。比纯朴素 CTA 更燃 |

**整篇关键学习点**

1. 极端式 Hook 用"低维度反差锚点"（我们村没通电）衬托"高维度事物"（阿帕奇）—— 杀伤力最强的写法
2. "来自未来的武器" = 武器拟人化为"穿越者"——金句修辞
3. 数据轰炸要具体到口径、数量、年份，不能笼统
4. "仅 X 就足以 Y"的小数字 vs 大数字对比是数字式 Hook 必备
5. 桥梁后 "从…到…" 平行排比让玩法A 1:1 对应铺垫段
6. CTA 前可加"燃料收尾句"提升整体情绪

**⚠️ 防过拟合提醒（这条样本你应该学什么 / 不应该照搬什么）**

| 可以抽象复用（学风格） | 绝对不可以照搬（这是该样本独有） |
|---|---|
| 低维度反差锚点法（找一个普通人能感知的"小"，对照高维度装备的"大"） | Hook 原句"它问世的时候，我们村甚至还没有通电"——**重灾区**，不要重复 |
| 数据轰炸句式（射速 / 服役年份 / 弹药数量 / 造价 / 杀伤力等） | "30 毫米链式机炮 / 16 枚地狱火 / 76 枚火箭弹"——这是阿帕奇专属参数 |
| "仅 X 就足以 Y" 小 vs 大对比写法 | "一架阿帕奇摧毁一个加强连"——阿帕奇专属 |
| 桥梁公式："而今在 [产品] 中" | "在你的掌中"是该样本的金句，可学但别原句搬 |

**生成新极端式 Hook 时**：找一个**和你产品相关的低维度反差锚点**，不要套"我们村没通电"。例（战火勋章可用变体）：
- ✅ "它服役时苏联还没解体，至今仍是美军主力（A-10 攻击机）"
- ✅ "它能用的弹药比一些国家的全部军费还贵（B-2 隐身轰炸机）"
- ✗ "B-2 出现的时候，我家还没有彩电"（机械模仿"我们村没通电"）

### 样本 5 · Samo_头_23 人类为何没有驯化老虎（跨题材代表）

**元信息**

| 项 | 值 |
|---|---|
| 产品 | 万龙觉醒 |
| 时长 | 45.0 秒 |
| CTR(7日) | 0.42% |
| 消耗 | ¥475,665.19 |
| 主情绪燃料 | 欲望落差（现实做不到 → 游戏里能做到） |
| Hook 类型 | B-1 揭秘式 + B-2 反转式 |
| 具体层主线词 | 驯化老虎 / 力量压制 |
| 抽象层主线词 | 现实不可能的，游戏能做到 |

**逐段标注**

#### 段 1 · Hook（0-7.4s）

> **口播**："有人问，为什么人类没有驯化老虎作为宠物，不是哥们，你见过一天要吃几十斤鲜肉，一言不合就能分分钟把你秒了的宠物吗"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 双钩子词："有人问"+"为什么" / ✅ 自带答案的悖论结构 / ✅ "不是哥们"网络热词介入 / ✅ 真世界画面（老虎特写） |
| 词库标签 | D-1：有人问、为什么、你见过 |
| 燃料嵌入 | 欲望落差种子（一个所有人都好奇但没想过的问题） |
| 风格质感学习点 | **"不是哥们"** 是网络梗+反问的合一。该号擅长把网络热词嫁接到知识叙事——降低门槛同时强化亲切感 |

#### 段 2 · 铺垫A+B（7.4-20.8s）

> **口播**："抛开饲养成本问题，驯化的前提是你要有足够的力量压制它，而你唯一能欺负老虎的机会，也就只有在它小的时候，等长大后，怕是连靠近它的勇气都没有。老虎作为独居且天性凶猛的顶级掠食者，就注定了难以被驯化"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 给出"驯化 = 力量压制"的关键概念 / ✅ "你唯一能欺负老虎的机会，也就只有在它小的时候"是金句 / ✅ "顶级掠食者"专业术语强化知识感 |
| 词库标签 | D-2：抛开…不说 / D-8：唯一…机会 |
| 燃料嵌入 | 欲望落差铺垫（用户在了解"为什么现实不可能"——为后面"游戏可能"做反差） |
| 风格质感学习点 | "怕是连靠近它的勇气都没有"——网络口语化（"怕是"）+ 具象动作（"靠近"）。该号铺垫段擅长口语+学术混合 |

#### 段 3 · 桥梁（20.8-28.5s）

> **口播**："但是在万龙觉醒里，这一切都不是问题。别说驯服百兽之王老虎了，就连比它大上百倍的远古巨熊，烈焰巨龙等灭世巨兽都不在话下"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 标准桥梁句"但是在 [产品] 里" / ✅ "这一切都不是问题"是欲望落差的关键释放点 / ✅ "比它大上百倍"是反转 + 维度升级双合一 |
| 词库标签 | D-3：但是在 X 里 / D-2：别说…就连…都 |
| 因果路径 | 现实痛点（驯化老虎需力量压制，做不到）→ 卖点机制（40 人团本团队力量压制巨兽）→ 玩家结果（不仅老虎，更大百倍的巨兽也行） |
| 风格质感学习点 | **"别说 X 了，就连 Y 都"** 是欲望落差的最强释放句式。该号特别善用——把目标抬高 100 倍而不是 1 倍 |

#### 段 4 · 玩法A（28.5-37.8s）

> **口播**："只要召集盟友进入 40 人巨兽团本，战士在前排扛伤，法师射手全力输出，灵活走位紧密配合，齐心协力击败它，就能将其收服成为你的战场宠物"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 因果回答（前面讲驯化需要力量压制，这里讲游戏里靠 40 人团本团队压制）/ ✅ 卖点真实（40 人团本、职业分工、收服系统）/ ✅ 具象动作（前排扛伤/全力输出/走位/收服） |
| 词库标签 | D-4：只要…就能 |
| 风格质感学习点 | 玩法A 描述要给"具体游戏过程"，不能只说"团队战斗"。"前排扛伤、全力输出、灵活走位"= 用户能在脑里看到的游戏画面 |

#### 段 5 · 玩法B（37.8-44s）

> **口播**："在联盟团战中召唤出来，简直就是移动的战争堡垒，瞬间碾压全场"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 维度跳跃（玩法A 收服巨兽 → 玩法B 巨兽在联盟团战中是战争堡垒）/ ✅ 跳跃方向：个体能力 → 体系战场感 |
| 词库标签 | D-5：碾压、瞬间 |
| 风格质感学习点 | "移动的战争堡垒"——把巨兽拟人化为军事单位概念。比"很强大"具体 |

#### 段 6 · CTA（44-45s）

> **口播**："一起来万龙觉醒驾驭巨兽，横扫魔兽万人战场吧"

| 维度 | 分析 |
|---|---|
| 段位规则点 | ✅ 含产品名 / ✅ 动词（一起来、驾驭、横扫）/ ✅ 22 字（≤25）/ ✅ "魔兽万人战场"对接核心受众（魔兽老玩家） |
| 词库标签 | D-6：一起来 / D-5：横扫 |
| 风格质感学习点 | CTA 含目标受众锚点（"魔兽"）—— 该游戏受众是 WCG 老玩家，这种锚点能强化下载意愿 |

**整篇关键学习点**

1. 网络热词嫁接知识叙事（"不是哥们"/"怕是"）—— 降低门槛 + 强化亲切感
2. **欲望落差释放句式**：但是在 X 里，这一切都不是问题。别说 [小目标] 了，就连 [大 100 倍目标] 都不在话下
3. 玩法A 必须给"具体游戏过程"，不能空话（"前排扛伤、走位、收服"）
4. 玩法B 维度跳跃可以走"个体 → 体系战场感"（巨兽 → 联盟团战的战争堡垒）
5. CTA 含目标受众锚点（"魔兽万人战场" 对接 WCG 老玩家）

**⚠️ 防过拟合提醒（这条样本你应该学什么 / 不应该照搬什么）**

| 可以抽象复用（学风格） | 绝对不可以照搬（这是该样本独有） |
|---|---|
| 钩子词组合"有人问 + 为什么"开场 | Hook 原句"为什么人类没有驯化老虎作为宠物"——直接复用 = FAIL |
| 反问句式"你见过 [极端特征] 的 X 吗"（强化欲望落差） | "一天要吃几十斤鲜肉、一言不合就能分分钟把你秒了的宠物"——这是老虎专属，别套到其他动物 |
| 网络热词嫁接（"不是哥们" / "怕是" / "骑脸"等） | 这些热词可以用，但不要每次都用同样几个，要换 |
| 桥梁句"但是在 [产品] 里" | "这一切都不是问题"是该样本特定话术，可学但别原句搬 |
| "别说 X 了，就连 Y 都" 欲望落差释放句式 | "百兽之王老虎 / 大上百倍的远古巨熊"——老虎对照专属，别套到非动物题材 |

**生成新欲望落差 Hook 时**：换一个**和你产品相关的现实痛点**，不要套"为什么人类没有驯化老虎"。例（万龙觉醒可用变体）：
- ✅ "为什么神话里的巨龙总让人类付出灭族代价"
- ✅ "如果地震是因为巨兽翻身，你敢去叫醒它吗"
- ✗ "为什么人类没有驯化霸王龙作为宠物"（句式抄太死，只是换对象）

### 样本 6 · ROK_头_05 东西方骑兵对比（E 类比式代表）

**元信息**：万国觉醒 / 57s / CTR 0.23% / 消耗 ¥36.8 万 / 民族优越感 / E 类比式 + B-1 揭秘式

**Hook**："为什么说对于骑兵的运用，东西方完全不是一个层次？这么说吧"

**Hook 学习点**：
- "为什么说对于 X，[我们/他们] 完全不是一个层次" 是 E 类比式标准开场——直接立判断而不是只抛问题
- "这么说吧" 是口语化承接，比"接下来我来讲"高级 10 倍

**关键金句**：
| 句 | 学习点 |
|---|---|
| "汉代霍去病以 1 万精锐骑兵，连续六日急行军，击破匈奴王庭指挥中枢的打法，是欧洲人难以想象的" | 具体数字（1 万 / 六日）+ 战术名（急行军、击破指挥中枢）+ 对比锚（欧洲人难以想象） |
| "1000 多年后，蒙古铁骑以快速机动、迂回包抄的东方战术横扫欧洲时，欧洲人称其为'上帝之鞭'" | 用"他们给我们起的外号"反衬威慑力——比自夸有说服力 100 倍 |
| "直到 1939 年，古德里安终于想到了，原来骑兵是这么用的。中国古代骑兵的作用，直到那时才有了个新名词，闪电战" | **教科书级类比式金句**：把"闪电战"这个西方军事术语揭示为"我们 X 千年前就在用"的东方智慧——民族优越感峰值 |

**桥梁**："所以在万国觉醒中，别看英式文明有亚瑟王的圆桌骑士加持，德式文明的条顿骑士还有攻击力加成，中式文明往往以诸葛弩火力强大而闻名"
**桥梁学习点**：用"别看 [他们的优势]"先承认西方文明优势，再反转引出中式——这种"先承认再反转"的桥梁比直接吹中式更有说服力

**玩法 B 维度跳跃**："曹操的虎豹骑还有 10% 的行军速度和 30% 的攻击力加成 / 再搭配诸葛亮的诸葛弩军，箭雨覆盖，横扫万国更是轻轻松松"——玩法A 单将（霍去病）→ 玩法B 多将组合（曹操+诸葛亮）+ 整体目标（横扫万国）

### 样本 7 · WGAME_头_11 轰炸机瞄准器（D 错觉反转代表）

**元信息**：战火勋章 / 51s / CTR 2.09% (本盘子最高) / 消耗 ¥32.9 万 / 智识优越感 / D 错觉反转

**Hook**："你以为这只是一个陀螺仪，但其实这是一个轰炸机瞄准器。更离谱的是，这玩意的造价竟堪比原子弹"

**Hook 学习点**：
- **"你以为 X，其实 Y"** 是 D 反转式最经典模板
- 紧跟"更离谱的是 + 第二反转（造价堪比原子弹）"—— 双层反转锁定情绪
- 用"陀螺仪"这个看似无关的对象做错觉锚——比直接说"瞄准器很贵"反差强 100 倍

**关键金句**：
| 句 | 学习点 |
|---|---|
| "用轰炸机投过弹的人都知道" | 装专家口吻——"投过弹的人都知道"暗示这是行内知识，强化智识感 |
| "据说它的造价高达十五亿美元" | "据说"+"高达"是数字式说服力技巧 |
| "之所以愿意投入如此巨资，仅仅是为了提升高空投弹的准确度" | "仅仅是为了"是反差句——巨资 vs 小事，反差越大震撼越强 |

**桥梁**："而承载这一核心任务的轰炸机在战火勋章中，同样是战场上的决胜力量"
**桥梁学习点**：用"承载这一核心任务的 [对象]"是个高级桥梁句式——把铺垫段讲的概念抽象成"核心任务"，再具象到游戏对象上

**玩法 B 维度跳跃**："还能越过敌方正面大部队直击后方 / 更能快速支援任何地点彻底摧毁敌方有生力量 / 通过空地协同灵活部署作战策略"——玩法A 大规模轰炸 → 玩法B 战术多样性 + 空地协同（从单纯火力 → 战略体系）

### 样本 8 · Samo_头_21 兽族为何没有终极空军（D 强反转代表）

**元信息**：万龙觉醒 / 48s / CTR 0.47% / **消耗 ¥147.8 万（盘子单条消耗冠军）** / 欲望落差 / D 强反转

**Hook**："有人问，为什么兽族没有自己的终极空军？仔细看兽族的兵力配置你就会明白"

**Hook 学习点**：
- "有人问 X？仔细看 Y 你就会明白" 是该号 D 反转式独特变体——先假装否定 Hook 的预设（你以为兽族没空军？让我告诉你）
- 结合"仔细看"的引导动作，让用户做出"我也要仔细看看"的代入

**关键金句**：
| 句 | 学习点 |
|---|---|
| "天生战种，兽族天生自带专属攻击力加成" | 概念命名（"天生战种"）+ 具体卖点（自带攻击力加成）—— 把游戏机制命名为种族特性 |
| "一波冲锋瞬间带走敌人半管血" | 玩家熟悉的"血量"词汇 + 具体动作—— 比"狼骑兵很强"有效 100 倍 |
| "飞龙骑脸直接把对面打到自闭" | 网络梗（"骑脸" / "打到自闭"）—— 适配 RTS 老玩家受众 |

**反转节点**："这么强的种族，要是再配个顶级空战单位，这简直就是开了挂的存在。其实在万龙觉醒中，这并非不能实现"
**反转学习点**：先讲"兽族陆军已经够强了"（呼应 Hook 的"为什么没有空军 = 不需要空军"），再反转"其实在 X 中能实现"—— 这种双重反转让欲望落差释放更强烈

**玩法 B 维度跳跃**："还能集结盟友进入 40 人巨兽团本，收服传说中的烈焰巨龙 / 它可不是普通的空中单位，而是真正能扭转战局的终极战争巨兽"——玩法A 兽族军团 → 玩法B 40 人团本巨龙（从兵种群 → 核心战略单位+战局）

## 6 条样本提炼出的极客号语言风格 8 条铁律

1. **悖论结构**：Hook 用"明明 X 为何 Y"或"你以为 X 其实 Y"或"为什么 [反常识]"，必须含反差
2. **金句修辞**：每段铺垫和桥梁有 1 个金句（不是抽象形容词，是具体场景/动作/数字组合）—— "我们村还没通电" / "迷人的老祖宗" / "上帝之鞭" / "移动的战争堡垒" / "低空死神" 等
3. **网络热词嫁接**：每条至少 1-2 个网络口语词嫁接到知识叙事（"不是哥们" / "怕是" / "骑脸" / "打到自闭" / "迷人的老祖宗"）—— 降门槛 + 拉近距离
4. **具体数字 vs 大数字对比**："仅 1 架阿帕奇就足以摧毁 1 个加强连" / "1 万精锐骑兵六日急行军" / "一匹马吃的比 3 个兵还多" —— 数字必须有对比锚
5. **逆向写法**："根本不用 X 都能 Y" / "仅仅是为了 X" / "甚至还没 X" / "怕是连 X 都没有" —— 比正向写法震撼 10 倍
6. **维度跳跃必有词**："不止 X 还 Y" / "更绝的是" / "真正的 X" / "一体化" / "全方位" / "横扫" / "碾压" —— 玩法B 必备
7. **桥梁朴素句**：永远是"然而在 X 中" / "而今在 X 中" / "但是在 X 里" / "所以当 X 在 X 中" —— 不要花哨的过渡
8. **CTA 短而具体**：含产品名 + 1 个动词 + 1 个核心体验词。绝不超 25 字。"史诗级"等空话坚决不出现
